# Spring

The [`Spring`](/guide/springs.md) class

## Methods

### `Get`

Gets the goal that was provided to the spring in the beginning.

**Returns**

- `any`

---
  
### `Stop`

Stops the spring from moving, and pauses any instances using it.

**Returns**

- `void`

---
  
### `Destroy`

Destroys the spring object.

**Returns**

- `void`
